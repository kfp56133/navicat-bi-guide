## Navicat BI 怎么用：从零到第一张图表

说完买的问题，说说用的问题。第一次打开 Navicat BI，界面里有三个核心概念：工作区、数据源、图表。理解这三层结构，就能上手了。

**第一步：新建工作区**

打开 Navicat BI，点击「新建工作区」，给它起个名字。工作区就是你的项目容器，所有数据源和图表都放在里面。一个工作区可以有多个数据源，一个数据源可以支撑多张图表。

**第二步：添加数据源**

在工作区窗口顶部点击「新建数据源」。你可以选择：
- 已有的数据库连接（MySQL、PostgreSQL 等，如果你装了 Navicat Premium 会直接列出已配置的连接）
- ODBC 数据源
- 外部文件（Excel、CSV、Access）

选好数据库后，可以直接写 SQL 查询，或者从 Navicat 导入已保存的查询语句。数据源配置好后，你的数据就准备好了。

**第三步：新建图表**

点击主工具栏「新建图表」，选择刚才配置的数据源，然后把数据字段拖拽到 X 轴、Y 轴对应位置。图表类型可以选条形图、折线图、饼图、散点图、热图、瀑布图、KPI 仪表盘……切换图表类型是实时预览的，拖一拖就知道哪种最能说明问题。

颜色、字体、透明度、趋势线都可以在右侧面板里调。不需要写一行代码。

**第四步：整合成仪表板**

图表做好了几张，点「新建仪表板」，把图表拖进来拼布局。可以加文字说明、背景图。仪表板上的图表如果来自同一个数据源，点击一张图表的某个数据点，其他联动图表会同步高亮过滤——这个交互效果在给非技术背景的人做展示时特别有用。

做好的仪表板可以保存为文件，也可以同步到 Navicat Cloud，邀请团队成员一起编辑。

## 常见问题解答

**Q：Navicat BI 和 Navicat Premium 里内置的 BI 功能有什么区别？**

Navicat BI 是专门为数据可视化设计的独立软件，功能更专注。Navicat Premium 里集成了 BI 功能模块，适合已经在用 Premium 做数据库管理的用户顺带可视化。如果你主要需求是做图表和仪表板而不是数据库管理，购买独立的 Navicat BI 授权更合适。

**Q：永久版的"子版本更新"是什么意思？**

大版本（如从 2.x 升级到 3.x）需要另行购买升级授权，子版本（如 2.1 升级到 2.2）在永久版授权期间免费更新。购买时可以参考当前的版本号来判断大版本升级的频率。

**Q：非商业版需要什么证明材料？**

一般需要提供能证明教育机构或非营利组织身份的文件，比如学校录取通知、教育邮箱或机构相关证明。具体要求可以查看 Navicat 官网的非商业版须知页面，购买前建议先确认自己的情况是否符合资格。

**Q：一个账号能装几台电脑？**

单个许可证支持激活 1 台设备。换电脑时，在旧设备上执行软件反激活操作，然后在新设备上重新激活即可，不用重复购买。

**Q：有没有更便宜的学生价？**
